DAC (цифрово аналоговый преобразователь). Выдаёт число от 0 до 4095 - аналоговый сигнал от 0 до 3.3V. SysTick - таймер. HSE - внешний источник тактирования. Предделители смотрим в .ioc в соседнем проекте (lab3_hal). PLL: после первого делителя получить 2 MHz, тогда будет проще считать, при изменении HSE будет меняться только первый предделитель. А множитель всегда выбираем равный желаемой SYSCLK.
SysTick тактируется от процессора. USART тут нужен чтоб менять частоту и амплитуду синуса.
