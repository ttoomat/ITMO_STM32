# ITMO_STM32

Проекты с названием labx:

Лабораторные работы по Программированию Микроконтроллеров STM32. Используемая плата: F446RE.

Выполнила Троицкая Тамара.

Осень 2025, 3 курс.

Остальное:

F103_ - моргание светодиодом на Bluepill

F411_DAC - использование внешней DAC на F411 (Blackpill). Летняя практика.

DC_drive_practice - недописанный проект с I2C для датчика тока. Летняя практика.

G474_AC_switch - управление двумя MOSFET через драйверы NSi6651A (в процессе). Летняя практика.
